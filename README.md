# flypea.tech

Portfolio and web development learning lab for `flypea.tech`.

## Stack

- Astro 7
- TypeScript
- Tailwind CSS v4
- Biome
- GitHub Actions

## Commands

```sh
npm install
npm run dev
npm run verify
```

## Development URL

```text
http://127.0.0.1:4321
```

## Deployment

Astro builds this site as static files into `dist/`.
The contents of `dist/` can be uploaded to Lolipop via SFTP/FTP.

GitHub Actions deployment will be added after the Lolipop SFTP credentials are available.
